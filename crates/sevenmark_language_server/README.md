# sevenmark-language-server

Language Server Protocol implementation for SevenMark wiki markup.

[![License](https://img.shields.io/badge/license-AGPL--3.0--only-blue.svg)](https://github.com/sevenwiki/sevenmark/blob/main/LICENSE)

## Features

- Diagnostics (parse errors, undefined variable warnings)
- Semantic tokens (57 token types)
- Go-to-definition for variables
- Hover information
- Completion (brace keywords, macros, variables)
- Folding ranges
